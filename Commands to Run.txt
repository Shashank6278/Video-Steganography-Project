WHAT IT MEANS

1. First command (extract original frame)

Extracts the first frame from original video
Used as baseline for comparison

2. Second command (extract stego frame)

Extracts the same frame from encoded (stego) video
Used to check changes after hiding data

3. Third command (PSNR)

Compares both images and measures quality difference
Higher value = images are almost identical

4. Fourth command (Python LSB + diff)

Counts how many pixels changed
Shows LSB bits to prove hidden data is stored there
RESULT: Pixel difference (284 pixels):

Only a few pixels changed out of thousands
This proves data is hidden with minimal modification


PSNR Full Form:
👉 Peak Signal-to-Noise Ratio

What PSNR values mean (simple):

> 50 dB → Almost identical (excellent, no visible difference)
40 – 50 dB → Very high quality (hard to notice difference)
30 – 40 dB → Good quality (slight difference)
20 – 30 dB → Poor quality (visible distortion)
< 20 dB → Very poor (major distortion)
